dominion-images by ivadla

Alternative card images which can be used with the card game dominion. Most of the images use cliparts from http://www.openclipart.org/

An overview of the cards is available here.

If you have any ideas for better images, you can open an issue at https://github.com/ivadla/dominion-images/issues or write an email to ivadla (see contact) The image should be Public Domain or have a license compatible with the Creative Commons Attribution-ShareAlike 3.0 License.
