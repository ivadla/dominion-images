dominion-images by ivadla

Alternative card images which can be used with the card game dominion. Most of the images use cliparts from http://www.openclipart.org/

An overview of the cards is available here.

If you have any ideas for better images, you can open an issue at https://github.com/ivadla/dominion-images/issues or write an email to ivadla (see contact).
The image should be Public Domain or have a license compatible with the Creative Commons Attribution-ShareAlike 3.0 License.

Card-generator

Most of the current images have been created with a Card-generator. This card-generator takes a template(svg), a card description (xml), an image for the card(svg, png or jpg) and a specification for the image and generates the card as a svg-file.

The old versions of the cards which were created manually can be found in the originalImages-branch

Authors

Adventuress: original author, did base, intrigue, seaside and alchemy (no longer active)
Enigmatosis: made prosperity cards without cardimages (no longer active)
ivadla: made cornucopia cards

The sources for the cliparts, which have been used to generate the cards, can be found in the sources.txt

Download

The GitHub download page has zip-files containing all the cards in jpg-format. That's probably what you are looking for.

Or you can download the whole project in either zip or tar formats. This includes svg, png and jpg formats of the cards as well as the templates and source cliparts, which have been used to generate the svg-files.

You can also clone the project with Git by running: 

$ git clone git://github.com/ivadla/dominion-images
